# 🚀 Network Security Scanning and Asset Discovery System (NinjaScanner)

An AI-powered cybersecurity system designed to scan networks, discover active assets, analyze open ports, and perform automated risk assessment for improved infrastructure security.

---

## 🧠 Overview

The **NinjaScanner** project provides an intelligent approach to network security by combining:

- Network scanning  
- Asset discovery  
- Port analysis  
- Risk classification  
- AI-based security insights  

This system helps identify exposed services and supports proactive cybersecurity risk management.

---

## 🔐 Key Features

- 🔍 Network scanning and asset discovery  
- 🌐 Full port scanning (1–1024)  
- 📊 Risk classification (HIGH / MEDIUM / LOW / SECURE)  
- 🧠 AI-based security analysis  
- 📄 Automated JSON report generation  
- 🖥️ Interactive dashboard (Streamlit)
